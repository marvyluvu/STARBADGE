 ## What is STARBADGE

<p>
An esp32-c3 mini based buisness card/ gift, It has some leds and looks super cool.
  its heavily inspired by the voyager golden records, The furthest human object in space ever.
</p>

---
##  Bill of Materials (BOM)

Key components

| Designator / Product         | Footprint / Description                                          | Qty |
| ---------------------------- | ---------------------------------------------------------------- | --- |
| U1 – ESP32‑C3 SuperMini_TH   | MODULE_ESP32-C3_SUPERMINI_TH – main MCU badge controller         | 1   |
| BZ1 – Voyager badge buzzer   | MLT‑8530 3–5 V SMD buzzer, 8.5×8.5 mm                            | 1   |
| D1, D2, D3 – badge LEDs      | LED_D5.0mm (any color you want!)                                 | 3   |
| R1, R2, R3 – LED resistors   | R_Axial_DIN0207_L6.3mm_D2.5mm_P7.62mm_Horizontal – 330 Ω         | 3   |
| R4 – buzzer resistor         | R_Axial_DIN0207_L6.3mm_D2.5mm_P7.62mm_P7.62mm_Horizontal – 220 Ω | 1   |
